# Project Management System

## Overview

A full-stack Project Management System built using React, Node.js, Express.js, PostgreSQL, Prisma ORM, and JWT Authentication.

## Features

* User Registration
* User Login
* JWT Authentication
* Dashboard Analytics
* Project Management
* Task Management
* Responsive User Interface

## Technology Stack

### Frontend

* React
* Vite
* React Router

### Backend

* Node.js
* Express.js
* Prisma ORM

### Database

* PostgreSQL

## Modules

* Authentication Module
* Dashboard Module
* Project Module
* Task Module
